# Websocket Datastore

usage: `wsapi [<flags>]`
```
Flags:
  -h, --help                 Show context-sensitive help (also try --help-long
                             and --help-man).
      --version              Show application version.
      --log.level="info"     Log level (all, debug, info, warn, error)
      --app.baseurl=""       App base url
      --app.user="api"       App user
      --app.password="api"   App password
      --env="prod"           Environment
      --datastore.path="data/ds.json"
                             Datastore path
      --listen.addr=":8080"  Listen addr for http api
```
